# 🧠 Problem Solving in JavaScript

## 📌 Overview

This repository is a structured collection of JavaScript problem-solving solutions covering:

* 📦 Frontend utility snippets
* ⚛️ React patterns and custom hooks
* 🧩 Machine coding & real-world problems

The goal is to build consistency, improve problem-solving skills, and write clean, production-ready code.
